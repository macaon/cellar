Cellar

A software storefront for Windows and Linux applications on the GNOME desktop.

1. Add the repository

flatpak remote-add --if-not-exists cellar https://macaon.github.io/cellar/index.flatpakrepo

2. Install Cellar

flatpak install cellar io.github.cellar

3. Run

flatpak run io.github.cellar

Updates

Once installed, flatpak update will pull the latest version automatically.
